Returns
Ritorna da 0 a 2, cioè il valore dove è presente una pedina diversa 0

Controllo sempre che non mi indichi una pedina al di fuori della scacchiera, cosi non va a controllare in posizioni non allocate della memoria.

if(a<0 || b<0 || a>=7 || b>=7) {
return 0;
}

Caso in cui è una casella vuota

if (scacchiera[a][b].composition[0] == 0) {
return 0;
}

Caso in cui in in posizione 1 non ci sia nulla e in posizione 0 ci sia un numero.

if (scacchiera[a][b].composition[1] == 0 && scacchiera[a][b].composition[0] != 0) {
return 0;
}

Caso in cui in in posizione 2 non ci sia nulla e in posizione 1 ci sia un numero.

if (scacchiera[a][b].composition[2] == 0 && scacchiera[a][b].composition[1] != 0) {
return 1;
}

Nel caso in cui non si soddisfino gli altri casi, cioè nel caso in cui ci sia un numero on composition[0] e composizion[1] e infine anche in compsosition[2].
